VERTICAL OVERLAYS — 1080 × 1920 (9:16)

For TikTok / YouTube Shorts / vertical streams in the green-neon theme. All scenes share an animated topographic background — flowing green contour lines that drift continuously.

OBS setup: Sources → + → Browser Source → Local file → pick a scene → set Width 1080, Height 1920, Custom FPS 60. For cam-frame / HUD scenes: layer your webcam source UNDER the Browser Source, positioned inside the glowing frame.
