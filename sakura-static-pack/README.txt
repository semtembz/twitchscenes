HOW TO USE — VIDEO (.webm)
──────────────────────────
In OBS / Streamlabs:  Sources → +  →  Media Source  →  pick the .webm.
  · Tick "Loop" for the Screens and Overlays.
  · Leave looping OFF for the Transition and Alerts (they play once).

STINGER TRANSITION:  Settings → Scene Transitions → + → Stinger →
  choose  Transition/Transition.webm  → set Transition Point ≈ 650 ms.

ALERTS:  add each alert .webm in your alert software (Streamlabs / StreamElements)
  as the alert's video/image. The clip shows the EVENT (e.g. "NEW FOLLOWER") —
  let your alert software print the viewer's NAME on top, so every alert shows
  the right person automatically.

WEBCAM FRAME:  put  Overlays/Webcam Frame.webm  ABOVE your camera source; your
  camera shows through the transparent middle.


HOW TO USE — IMAGES (.png)
──────────────────────────
  Offline        Twitch → Settings → Channel → Brand → "Video Player Banner"
  Profile Banner Twitch → Settings → Channel → Brand → "Profile Banner"
  Panels         Your channel → About → Edit Panels → + (upload each)
  Icons          use them in your panels, description, or socials


NOTES
